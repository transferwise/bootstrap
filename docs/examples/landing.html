Landing Pages

Landing pages use a jumbotron with background imagery. The imagery should have a focal point on the right hand side of the page

This is a fork of the main Bootstrap project implementing the TransferWise style guide.

Large buttons

Landing pages are split into colour blocked sections

Text is centred and the colour of text is reactive to the background colour. The first section following a jumbotron should be have a white or grey background. The title of a section uses h2.

A landing page will almost always contain a Triptych

Triptychs have 3 columns and normally feature iconography. If preceded by text it should use lead. Triptychs should feature the approximately the same amount of text in each column.

Landing Template

Our landing template features a jumbotron and a number of sections.

Landing templates should be used where we expect a user to first arrive on the site, or when entering a new section of the site, e.g. business

Create Template

The create template is used for forms and where we expect a user to create something. It uses a two column layout with a large panel to the left and navigation or secondary information in the right hand column.
